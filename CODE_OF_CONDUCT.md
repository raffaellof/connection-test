# Contributor Code of Conduct

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

## Attribution

Project maintainers are responsible for clarifying and enforcing this code of conduct. Violations may result in warnings, temporary bans, or permanent removal from the project.

## Enforcement

If you experience or witness unacceptable behavior, please report it by opening an issue or contacting a maintainer directly. All reports will be handled confidentially.

## Reporting Issues

- Disruptive or aggressive behavior in public or private communication.
- Threats, stalking, or unwelcome attention.
- Offensive comments related to gender, sexual orientation, disability, ethnicity, religion, or personal characteristics.
- Harassment, intimidation, or discrimination in any form.

## Unacceptable Behavior

- Help create a collaborative and supportive environment for all.
- Respect the decisions of maintainers and the community.
- Avoid personal attacks, insults, or discriminatory language.
- Value constructive feedback and accept differing viewpoints.
- Be respectful and considerate in all interactions.

## Our Standards

As contributors and maintainers of the **connection-test** project, we pledge to foster an open, welcoming, diverse, inclusive, and harassment-free environment for everyone.
